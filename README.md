# 블로그 프로젝트
- 개인블로그 컨셉의 웹 어플리케이션 프로젝트입니다.
- 백엔드 - https://github.com/hyeok-min/blog_project
- 프론트엔드 - https://github.com/hyeok-min/blog-project-front
- 세부구현기능
   - 마이페이지
   - 게시글 CRUD
   - 댓글과 대댓글
   - 문의사항(관리자, 클라이언트 별도)

# 개발환경
- Spring Boot
- Spring-Data-Jpa
- JDK 17
- Mysql
- Query-DSL
- React
